# Ops360 Overview

Ops360 is a unified operations management platform designed to streamline monitoring, troubleshooting, and automation across cloud and hybrid environments. It provides a centralized experience for operational insights, agentic workflows, and seamless integration with Azure services.

## Key Features
- Unified dashboard for operational data
- Agentic experience for guided troubleshooting
- Integration with Azure Monitor and other tools
- Scalable onboarding and licensing management

## Audience
Ops360 is intended for IT operations teams, cloud administrators, and service owners seeking to improve efficiency and visibility across their environments.
