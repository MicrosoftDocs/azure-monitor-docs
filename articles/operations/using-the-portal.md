# Using the Azure Operations Center Portal

The Ops360 portal is your central hub for managing operations, monitoring resources, and accessing agentic workflows.

## Portal Features
- Unified dashboard for all operational data
- Customizable views and widgets
- Access to agentic guidance and automation tools
- Role-based access control

## Navigation
- Home: Overview of key metrics and alerts
- Agentic Experience: AI-driven troubleshooting and automation
- Licensing: Manage and review license usage
- Settings: Configure integrations and user roles

## Tips
- Use search and filters to quickly find resources
- Pin frequently used widgets to your dashboard
- Review notifications for important updates

## Help & Support
Access documentation, FAQs, and support directly from the portal menu.
