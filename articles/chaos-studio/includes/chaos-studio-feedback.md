## Report issues and request features

Azure Chaos Studio is developed in the open. To report a bug, request a feature, or ask a question about Workspaces, Scenarios, or the Azure CLI extension, open an issue in the [Chaos Studio repository on GitHub](https://github.com/microsoft/chaos-studio/issues). By filing an issue, you can track its progress and see requests from other customers.
