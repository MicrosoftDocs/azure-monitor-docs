# Azure Monitor overview page

The default **Monitor** overview page in the Azure portal includes descriptions and links for different features of Azure Monitor. You can click on each of the links for more information about the feature or to jump to the feature in the Azure portal. 

:::image type="content" source="./media/overview-page/monitor-overview.png" lightbox="./media/overview-page/monitor-overview.png" alt-text="Screenshot of new Azure Monitor overview page in the Azure portal.":::

Select **New overview** to use the new overview experience which surfaces high level issues and recommendations across your Azure environment. 

:::image type="content" source="./media/overview-page/monitor-overview-new.gif" lightbox="./media/overview-page/monitor-overview-new.gif" alt-text="Screenshot of Azure Monitor overview page in the Azure portal.":::


The new overview experience includes the following sections:


| Section | Description |
|:---|:---|
| Top actions | Top recommended actions based on Azure Advisor recommendations to ensure that your resources are being appropriately monitored. Click on any of these actions to open the relevant page in Azure Monitor. |
| Monitoring status | Identifies any recent service incidents from **Azure service health** and a summary of [issues](/azure/azure-monitor/aiops/aiops-issue-and-investigation-overview) created over the previous 24 hours.  |
